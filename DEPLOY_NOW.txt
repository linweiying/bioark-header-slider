=================================================================
  BioArk Header Slider - 立即部署指令
=================================================================

✅ 项目已完成！现在只需 3 步即可上线：

-----------------------------------------------------------------
步骤 1️⃣: 创建 GitHub 仓库
-----------------------------------------------------------------

1. 打开浏览器访问: https://github.com/new

2. 填写以下信息:
   Repository name: bioark-header-slider
   Description: 响应式 Header 轮播组件 - Vue 3 + TypeScript
   选择: Public (公开)
   
   ⚠️ 重要: 不要勾选 "Add a README file"

3. 点击 "Create repository"

4. 复制页面上显示的仓库 URL，例如:
   https://github.com/你的用户名/bioark-header-slider.git

-----------------------------------------------------------------
步骤 2️⃣: 推送代码到 GitHub
-----------------------------------------------------------------

在命令行中运行以下命令（替换 YOUR_USERNAME 为你的 GitHub 用户名）:

cd e:\zuomian\aizhi\bioark-header-slider

git remote add origin https://github.com/YOUR_USERNAME/bioark-header-slider.git

git push -u origin main

-----------------------------------------------------------------
步骤 3️⃣: 部署到 Vercel（两种方式任选其一）
-----------------------------------------------------------------

【方式 A - 网页部署（推荐，更直观）】

1. 访问: https://vercel.com
2. 使用 GitHub 账户登录
3. 点击 "Add New..." → "Project"
4. 选择 "bioark-header-slider" 仓库
5. 点击 "Import"
6. 直接点击 "Deploy"（配置会自动检测）
7. 等待 1-2 分钟
8. ✅ 获得预览链接: https://bioark-header-slider.vercel.app

【方式 B - 命令行部署（更快）】

在命令行中运行:

cd e:\zuomian\aizhi\bioark-header-slider

vercel login
(会打开浏览器进行登录)

vercel --prod
(按提示选择默认选项)

✅ 完成后会显示预览链接
